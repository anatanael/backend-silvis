# WhatsApp Backend REST + Evolution API

Backend REST desenvolvido com Node.js + Express integrado à Evolution API.

## Funcionalidades

- Gerar QR Code do WhatsApp
- Verificar status da instância
- Listar chats
- Buscar chat específico
- Estrutura organizada
- Tratamento global de erros
- Variáveis de ambiente
- Respostas padronizadas em JSON

---

# Tecnologias

- Node.js
- Express
- Axios
- Dotenv
